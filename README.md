# 🌤️ Thimphu Weather App

A simple weather app that shows real-time weather for Thimphu, Bhutan.

## Features

- 🌡️ Shows current temperature
- 💨 Shows wind speed
- ☁️ Shows weather condition (Clear, Rain, Cloudy)
- 🎨 Beautiful window with buttons
- 💾 Save weather to log file

## How to Use

### Option 1: With Python
```bash
python thimphu_weather_app.py
```

### Option 2: EXE File (No Python needed)
Just double-click `thimphu_weather_app.exe`

## Requirements

- Python 3.8 or higher
- Library: `requests`

## Installation

```bash
pip install requests
```

## Tech Stack

- **Language:** Python
- **Interface:** Tkinter (Windows)
- **API:** Open-Meteo (Free weather API)
